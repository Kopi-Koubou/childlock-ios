# Childlock — Screen-Time Breaks Through Active Challenges

**Created:** 2026-03-12
**Project ID:** childlock

## One-line idea

An iOS parental control app that turns passive screen time into active micro-learning moments. Instead of just limiting screen time, Childlock interrupts children’s YouTube/TikTok/etc sessions at fixed intervals (eg 10 min) and requires the child to complete an age-appropriate challenge (math, memory, pattern, motor challenge) to unlock the next chunk.

## Core problem

Children can enter long passive loops on short-form/video apps. Parents need better tools than blunt "cut it off for X mins" controls. Existing lock apps don’t optimize for cognitive engagement or child development.

## Why it’s better

- **Active engagement over passive restriction:** Kids earn screen time by solving quick challenges.
- **Habit building:** Creates interruption rhythm (think healthy cognitive breaks, not just punishment).
- **Parent trust loop:** Parents control app targets, intervals, difficulty, and review completion.
- **Scalable behavior pattern:** Can expand from home use (parents) to school/daycare (institutional controls) later.

## Core flow

1. Parent selects protected apps + interval (eg 10 mins)
2. Child uses device
3. At interval, interface locks/blocks and launches mini-challenge
4. Successful completion unlocks next screen-time block
5. Parent sees streaks, completion, and usage summaries

## Target user

- **Primary (B2C):** Parents with children ages 3-12
- **Secondary (future):** Schools/daycares for controlled device environments

## MVP feature set

- Parent setup: app list, interval duration, age band, allowed duration
- Challenge engine: math, memory, visual selection tasks by age band
- Unlock flow: pass challenge to continue
- Parent dashboard: usage and completion summary
- Free tier with limited daily challenge unlocks, premium unlocks unlimited + deeper insights
